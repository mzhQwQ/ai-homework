本项目使用**回溯法**求解 N 皇后问题：
1.  逐行放置皇后，保证每行仅放 1 个皇后；
2.  每次放置前，通过 `is_safe` 函数检查当前位置是否与已放置皇后**同列、同左上对角线、同右上对角线**；
3.  若位置合法，则放置皇后并递归处理下一行；若递归到最后一行，则记录为一个合法解；
4.  若当前行无合法位置，则回溯至上一行，撤销当前皇后并尝试下一列。运行方式
